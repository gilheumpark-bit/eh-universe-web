## 1. 결론

이 상용 패키지는 연구 성과를 포함한다.  
또한 원본도 `baseline/source of truth`로 포함한다.

핵심 계보:

```text

-> 1~100 연구 계획/실증 산출
-> 앱 전용 NOA Brain 설계
-> NOA Studio Brain 상용 제품 설계
```

## 4. 원본 대비 제품화 변화

| 원본/연구 단계 | 상용 제품 단계 |

| 연구 산출물 축적 | 제품 기능/데모/백로그로 변환 |
| AI 뇌구조 설계 | 앱 런타임 정책/라우터/receipt 모듈 |

## 5. 제품에 들어간 연구 기능

상용 설계에 실제로 흡수된 연구 기능은 다음이다.

```text
1. 탭별 전문가 라우팅
2. 관점 융합 기반 전문가 조합
3. 적용 전 위험도 판단
4. 적용 준비도 판단
5. 대량 변경 분할
6. 원고/설정 손상 보호
7. 작업 영수증
8. 복구 힌트
9. 출고 전 보호
10. 품질 게이트와 상용 준비도 등급
```

## 6. 제품에 아직 안 들어간 연구 기능

아직 제품화되지 않은 항목도 명확히 구분한다.

| 항목 | 현재 상태 | 이유 |
|---|---|---|
| 완전 자율 연구원 | 미적용 | 앱 제품에서는 위험도가 높음 |
| 외부 Judge 자동 평가 | 미적용 | 비용/품질/책임 경계 필요 |
| RAG 자동 수집/재가공 런타임 | 설계만 있음 | 데이터 정책과 저장 구조 필요 |
| 수식 자동 보정 루프 | 미적용 | 반복 평가 데이터 필요 |
| 물리/양자 법칙 기반 시뮬 연구실 | 제품 범위 밖 | 창작 앱의 1차 판매 가치와 거리가 있음 |

## 7. 판매 시 허용되는 주장

허용:

```text
NOA Studio Brain은 WABI-R/NOA 연구 프레임워크를 앱 제품에 맞게 축소·전문화한 창작 관제 설계다.
```

허용:

```text
탭별 전문가, 안전 적용, 작업 영수증, 출고 보호를 통해 AI 창작 보조의 위험을 줄이도록 설계되었다.
```

조건부 허용:

```text
실제 앱 구현과 베타 검증 이후에만 "제품에서 작동한다"고 말할 수 있다.
```

금지:

```text
완전 자율 AI 연구원이다.
상업 출고의 법적 안전성을 보장한다.
작품 손상을 완전히 막는다.
성능 개선률이 실증 없이 확정되었다.
```
