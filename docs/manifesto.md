## 🏛 2대 기둥 선언

### 기둥 ①: 카테고리 선언 (시장 구분)

> **"해줘는 컨텐츠 소비. 하라는 제작이다."**

AI가 대신 써주는 플랫폼(Sudowrite/Novelcrafter 등)은 "해줘" 시장 = **컨텐츠 소비 도구**.
작가가 도구로 직접 제작하는 플랫폼(Adobe/VS Code/Loreguard)은 "하라" 시장 = **제작 도구**.
**두 시장은 다른 산업이다.**

### 기둥 ②: 자유-책임 선언 (관계 구분)

> **"자유는 우리가. 책임은 당신이."**

플랫폼은 창작의 자유를 제공한다 (도구 중립성).
작가는 자기 창작물의 주인이자 책임자 (저작권 100% 귀속 + 플랫폼 면책).
이 관계가 Adobe·VS Code 계열 제작 도구의 본질이다.

## 🌐 4언어 카피 라이브러리

### 기둥 ① 카테고리 선언

```
KO: 해줘는 컨텐츠 소비. 하라는 제작이다.
EN: "Do it for me" is consumption. "Let me do it" is creation.
JP: 「やって」は消費、「やらせろ」は制作。
CN: "帮我做"是消费。"让我做"是创作。
```

### 기둥 ② 자유-책임 선언

```
[풀 버전]
KO: Loreguard는 작가에게 창작의 자유를 드립니다. 작가는 그 창작에 책임을 다 하시면 됩니다.
EN: Loreguard gives writers the freedom to create. Writers take responsibility for what they create.
JP: Loreguard は作家に創作の自由を。作家はその創作に責任を持てば十分。
CN: Loreguard 赋予作家创作自由。作家只需对自己的创作负责。

[압축 버전]
KO: 자유는 우리가. 책임은 당신이.
EN: Your freedom. Your responsibility.
JP: 自由は私たちが。責任はあなたが。
CN: 自由由我们给予。责任由您承担。
```

### 포지셔닝 (소설가 IDE)

```
KO: 소설가를 위한 AI IDE — 집필·검수·번역·출간을 한 곳에서
EN: The AI IDE for Novelists — Write, Review, Translate, Publish
JP: 小説家のための AI IDE — 執筆・校正・翻訳・出版をひとつに
CN: 小说家的 AI IDE — 创作·审阅·翻译·出版一站式
```

### 번역가 스타화

```
KO: AI는 번역가를 죽이지 않는다. 평범한 번역가를 죽이고, 프로 번역가를 스타로 만든다.
EN: AI doesn't kill translators. It kills mediocre ones and turns pros into stars.
JP: AIは翻訳者を殺さない。凡庸な者を淘汰し、プロをスターにする。
CN: AI 不会杀死译者。它淘汰平庸者，让专业人士成为明星。
```

### 출판사 DIY

```
KO: 외국 플랫폼이 발견하기를 기다리는 시대는 끝났다. 이제 한국 출판사가 직접 세계로 간다.
EN: Waiting to be discovered is over. Korean publishers now take their works global directly.
JP: 海外プラットフォームに発見される時代は終わった。韓国の出版社が直接世界へ。
CN: 等待海外平台发现的时代结束了。韩国出版社现在直接走向世界。
```

### 얼리 액세스 (알파)

```
KO: 알파 작가 50명 = 기간 한정 할인 + 알파 기여자 명시 = 얼리 액세스 멤버
EN: 50 alpha writers = limited-time discount + alpha contributors = early-access members
JP: アルファ作家50名 = 期間限定割引 + アルファ貢献者記載 = アーリーアクセスメンバー
CN: 50 位测试作家 = 限时折扣 + 测试贡献者署名 = 早期访问成员
```

### 양파 침투

```
KO: 심사장 없음. 시장에 양파만 있음. 껍질 까는 놈이 다 먹는다.
EN: No audition stage. Only onions in the market. Those who peel, eat.
JP: 審査場はない。市場にはタマネギだけ。皮をむく者が全てを得る。
CN: 没有选秀舞台。市场上只有洋葱。剥皮者得一切。
```

## 🔬 제작 철학 (내부 원칙)

### 3-Persona [C/G/K]
- **[C] 안전성**: null 가드 / 예외 처리 / 타입 / 보안
- **[G] 성능**: O(n²) 회피 / 불필요 복사 / 배치 처리
- **[K] 간결성**: 미사용 제거 / DRY / 데드코드 금지

### 효율 금지 원칙
> "파일 1개당 Read 강제. 통계 기반 요약 금지. 돌 깎는 장인."
- grep 스킵 안 됨
- 깨끗한 파일도 진단 기록
- 3루프 반복 검증

### 움직이는 제품 원칙
> "라이브 시연이 PPT보다 1000배 가치"
- 심사장에서 실시간 번역 시연
- GitHub 공개 코드 = 검증 가능한 증거
- 작가에게는 30초 체험 위젯
