## 🔧 검증 방법

### 수동 버전
- 브라우저에서 직접 클릭/입력/확인
- 각 항목 옆에 ✅ PASS / ❌ FAIL / ⚠️ WARN 기록
- FAIL 시: 스크린샷 + 콘솔 에러 + 재현 경로 기록

### 자동 버전 (Claude Code Preview / Playwright)
- `preview_start` → `preview_click` → `preview_snapshot` → `preview_console_logs` 순서
- 각 항목을 스크립트로 실행, DOM 상태 + 콘솔 에러 자동 수집
- FAIL 판정: 콘솔 error 존재 OR 기대 DOM 요소 부재 OR 클릭 후 상태 미변경
