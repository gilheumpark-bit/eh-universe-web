# Phase 1 — P0 스모크 (30~60분)

릴리스·주요 머지 후 **한 번** 돌립니다. 전부 체크되면 P0 통과로 간주합니다.

## 환경

- [ ] 데스크톱(Electron 또는 `npm run dev` 렌더러) 기준
- [ ] DevTools 콘솔 열림(심각 에러 확인)

## 핵심 루프

- [ ] 프로젝트/파일 열림(또는 데모/샘플)
- [ ] 에디터에 문자 입력 후 더티 표시 또는 저장 흐름 확인
- [ ] 탭 전환 1회 이상

## 우측 패널 (필수 샘플)

레지스트리 `isEssential` 패널 중 최소 아래를 **열기 → 닫기** (동작 확인):

- [ ] `chat`
- [ ] `search`
- [ ] `preview`
- [ ] `pipeline`
- [ ] `git`

## 에디터 컨텍스트 메뉴

- [ ] 에디터 본문 우클릭 → 메뉴 표시
- [ ] 잘라내기 / 복사 / 붙여넣기 / 문서 서식 / 모두 선택 — 각 1회 또는 대표 샘플
- [ ] **에디터 명령…** — Quick Command 또는 F1 경로 반응
- [ ] **스튜디오 명령 팔레트** — 앱 팔레트 열림

## 명령·단축키

- [ ] `Ctrl+Shift+P` — 스튜디오 명령 팔레트

## 콘솔

- [ ] 빨간 스택 트레이스 없음(경고만은 팀 정책에 따름)

## 실패 시

`notes`에 재현 단계·스크린·브랜치명 기록 후 이슈 등록. P0는 릴리스 차단 여부를 팀 규칙으로 결정.
