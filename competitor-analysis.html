5. EH Universe SWOT 분석

💪 강점 (Strengths)

• 유일한 "서사 엔진" 보유 — 장르 규칙, 텐션 모델링, Shadow State를 코드 레벨로 구현한 플랫폼은 시장에 없음
• BYOK로 구독료 제로 — API 비용만 지불, 월 $2~5 수준으로 전문 도구 사용 가능
• 한국 웹소설 문법 내장 — 고구마/사이다, 도파민 디바이스, 클리프행어 등 한국 독자 리텐션 공식 직접 반영
• 5사 15+모델 선택 자유 — 경쟁사 대비 압도적 AI 모델 다양성
• 연출 스튜디오 13탭 — 씬 단위 연출 제어는 업계에서 가장 세밀

⚠️ 약점 (Weaknesses)

• 커뮤니티 & 사용자 기반 미확보 — 아직 얼리 스테이지, 레퍼런스 작품 없음
• 집필 UX 완성도 — Sudowrite 대비 에디터 경험이 아직 부족 (포커스 모드, 단축키 등)
• 이미지 생성 없음 — NovelAI의 애니 일러스트, 뤼튼의 이미지 생성 같은 시각 요소 부재
• 기능 깊이의 양면성 — 13탭·5단계 규칙 등 고급 기능이 많지만, 모든 탭에 AI 자동 생성이 있어 기본 사용은 원클릭 수준. 다만 "기능이 많아 보이는" 첫인상이 진입장벽으로 작용할 수 있음
• 서버리스 구조 한계 — Google Drive 동기화만으로는 협업/백업 안정성 부족

🚀 기회 (Opportunities)

• 한국 웹소설 시장 1.2조원+ — AI 집필 도구 침투율 아직 <5%, 성장 여지 거대
• 플랫폼(카카페/문피아) 연동 — 분량/포맷 자동 맞춤 기능으로 작가 직접 연결 가능
• 장르 프리셋 확장 — 로맨스판타지, BL, GL 등 세분화로 니치 장악
• 아시아 시장 확장 — JP/CN i18n 이미 구현, 일본 나로우/중국 기점으로 확장 용이
• EH 룰북 자체가 IP — 교육/워크숍/출판사 컨설팅 등 B2B 사업 가능

🔥 위협 (Threats)

• ChatGPT/Claude/Gemini 직접 사용 — "그냥 챗봇에 쓰면 되잖아" 인식 극복 필요
• Sudowrite의 한국어 지원 가능성 — Muse 모델이 다국어 확장 시 직접 경쟁
• 플랫폼사 자체 AI 도구 — 카카오페이지/문피아가 자체 AI 집필 도구 출시 가능성
• AI 규제 — 한국 AI 저작권법 강화 시 AI 생성 콘텐츠 플랫폼 등록 제한 가능
• API 비용 변동 — Gemini/OpenAI 가격 인상 시 BYOK 모델의 비용 이점 축소

6. EH Universe만의 핵심 차별점

1️⃣ 서사 엔진 ≠ AI 챗봇

다른 도구들은 "AI에게 글 써달라" 수준. EH Universe는 규칙 레벨(Lv1~5), 텐션 수학 모델, Clean Taste 밸런스 등 구조적으로 서사 품질을 제어하는 유일한 플랫폼.

2️⃣ 한국 웹소설 DNA

고구마/사이다, 도파민 디바이스, 회차별 분량 최적화(모바일/웹), 상표 자동 필터 — 한국 웹소설 플랫폼에 "바로 연재 가능한" 수준의 출력을 목표로 설계.

3️⃣ BYOK = 구독료 제로

Sudowrite $10~59/월, NovelAI $10~25/월 vs EH Universe는 자기 API 키만 있으면 무료. Gemini Flash 사용 시 월 $1~3 수준으로 전문 서사 엔진 사용 가능.

7. 종합 결론

EH Universe는 "서사 엔진"이라는 카테고리 자체를 새로 정의한 제품입니다. 기존 AI 창작 도구들이 "AI가 글을 써줌" 수준에서 머무르는 반면, EH Universe는 장르 규칙, 텐션 모델링, 캐릭터 연속성, 독자 몰입도 예측까지 구조적으로 서사 품질을 제어합니다.

특히 한국 웹소설이라는 명확한 타겟 시장에서, 고구마/사이다 구조, 도파민 디바이스, 클리프행어 설계 등 장르 문법을 코드 레벨로 구현한 도구는 전 세계에 이것뿐입니다. BYOK 모델로 구독료 없이 사용 가능하다는 것도 강력한 차별점입니다.

단, 커뮤니티 부재와 UX 완성도, 기능 복잡도에 따른 진입장벽은 초기 성장의 병목이 될 수 있습니다. "파워 유저에게는 꿈의 도구, 일반 사용자에게는 학습 곡선"이라는 양면성을 어떻게 해결하느냐가 다음 단계의 핵심 과제입니다.
